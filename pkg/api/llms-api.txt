# dnsall.com API

> Automated SSL certificate DNS verification service. Manage ACME DNS-01 challenges via HTTP API.

Endpoint: https://api.dnsall.com
Dashboard: https://dnsall.com

## Quick Start

1. Register at https://dnsall.com or via API
2. Add your domain — get a CNAME record to configure
3. Set the CNAME in your DNS
4. Use lego/Traefik with httpreq provider to issue certificates

## Authentication

- **Web API**: POST /api/login → JWT token, use as `Authorization: Bearer {token}`
- **httpreq API**: HTTP Basic Auth with `username` and `api_key` (not password)

## Endpoints

### Register & Login

```
POST https://api.dnsall.com/api/register
Content-Type: application/json
{"username": "myuser", "password": "mypassword"}
→ {"token": "jwt...", "username": "myuser", "api_key": "abc123..."}

POST https://api.dnsall.com/api/login
Content-Type: application/json
{"username": "myuser", "password": "mypassword"}
→ {"token": "jwt...", "username": "myuser", "api_key": "abc123..."}
```

### Manage Domains (JWT required)

```
GET https://api.dnsall.com/api/domains
Authorization: Bearer {token}
→ [{"domain": "example.com", "subdomain": "182821e6", "cname_target": "182821e6.s.dnsall.com"}]

POST https://api.dnsall.com/api/domains
Authorization: Bearer {token}
{"domain": "example.com"}
→ {"domain": "example.com", "subdomain": "182821e6", "cname_target": "182821e6.s.dnsall.com"}

DELETE https://api.dnsall.com/api/domains/example.com
Authorization: Bearer {token}
→ 204
```

### View Records & Profile (JWT required)

```
GET https://api.dnsall.com/api/records
Authorization: Bearer {token}
→ [{"domain": "182821e6.s.dnsall.com", "value": "challenge-token", "last_update": "..."}]

GET https://api.dnsall.com/api/profile
Authorization: Bearer {token}
→ {"username": "myuser", "api_key": "abc123..."}

POST https://api.dnsall.com/api/profile/regenerate-key
Authorization: Bearer {token}
→ {"api_key": "newkey..."}
```

### DNS Challenge (Basic Auth: username + api_key)

```
POST https://api.dnsall.com/present
Authorization: Basic base64(username:api_key)
{"fqdn": "_acme-challenge.example.com.", "value": "challenge-token"}
→ 200

POST https://api.dnsall.com/cleanup
Authorization: Basic base64(username:api_key)
{"fqdn": "_acme-challenge.example.com.", "value": "challenge-token"}
→ 200
```

### Server Info (no auth)

```
GET https://api.dnsall.com/api/info
→ {"base_domain": "s.dnsall.com", "api_domain": "api.dnsall.com"}
```

## CNAME Setup

After adding a domain, set this DNS record:

```
_acme-challenge.example.com  CNAME  182821e6.s.dnsall.com
```

The subdomain (182821e6) is deterministic per user+domain and shown in the API response.

## lego Usage

```bash
LEGO_DISABLE_CNAME_SUPPORT=true \
HTTPREQ_ENDPOINT=https://api.dnsall.com \
HTTPREQ_USERNAME=myuser \
HTTPREQ_PASSWORD=<api_key> \
lego --dns httpreq --dns.propagation-disable-ans \
  --domains example.com --domains "*.example.com" \
  --email admin@example.com --accept-tos run
```

## Traefik

```yaml
# traefik.yml
certificatesResolvers:
  letsencrypt:
    acme:
      email: admin@example.com
      storage: /data/ssl/acme.json
      dnsChallenge:
        provider: httpreq
        propagation:
          disableChecks: true

# environment
LEGO_DISABLE_CNAME_SUPPORT=true
HTTPREQ_ENDPOINT=https://api.dnsall.com
HTTPREQ_USERNAME=myuser
HTTPREQ_PASSWORD=<api_key>
```

## Error Codes

- `unauthorized` — missing or invalid credentials
- `invalid_token` — expired or invalid JWT
- `domain_not_authorized` — user doesn't own this domain
- `domain_already_exists` — domain already added by this user
- `username_taken` — username already registered
- `invalid_credentials` — wrong username or password
- `missing_fqdn_or_value` — present/cleanup missing required fields
